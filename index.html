DSGO Stelselvoorzieningen

  1. DSGO Developer portal

    Website

    Het Developer Portal is een online interface waar ontwikkelaars documentatie, tools, en middelen kunnen vinden om effectief gebruik te maken van de DSGO specificaties, waardoor ze gemakkelijk softwaretoepassingen kunnen bouwen en integreren.

  2. DSGO Conformiteits Test Tool

    GitHub

    Een Conformiteits Test Tool (CTT) is een softwarehulpmiddel dat is ontworpen om de DSGO beheerorganisatie (i.o.) te laten verifiëren of een API voldoet aan de gestelde DSGO specificaties en standaarden.

  3. DSGO Stelselcatalogus

    Website API Docs

    De Stelselcatalogus biedt zichtbaarheid van DSGO datadiensten, zowel voor mensen (website) als machines (via APIs)
